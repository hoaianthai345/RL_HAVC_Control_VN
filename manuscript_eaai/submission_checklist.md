# EAAI Submission Checklist

- [ ] Replace all placeholder results in `main.tex` with measured values.
- [ ] Remove author-identifying information from anonymized manuscript.
- [ ] Prepare separate title page with author names and affiliations.
- [ ] Use single-column formatting.
- [ ] Keep manuscript under 50 pages and under 100 MB.
- [ ] Expand all acronyms in title and abstract.
- [ ] Include 3 to 5 highlights, each under 85 characters.
- [ ] Add graphical abstract if available.
- [ ] Add data availability statement.
- [ ] Add declaration of generative AI use if AI tools supported drafting.
- [ ] Add competing interest declaration.
- [ ] Add funding statement.
- [ ] Verify every reference manually before submission.
- [ ] Archive code/configs/results and include anonymized link if possible.
